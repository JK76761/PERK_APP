# App Store Screenshot Plan

Last updated: 2026-05-20

## Recommended first set

Prepare at least these screens for iPhone screenshots:

1. Home
   - show hero
   - verified badge
   - hot right now cards

2. Explore
   - show trending this week
   - category chips
   - collections

3. Detail
   - show perk hero
   - verified section
   - official source row

4. Saved
   - show saved perks list

5. Profile / Support
   - show trust and support options

## Suggested captions

1. `Discover verified ways to save`
2. `Explore food, banking, tools, and rewards`
3. `See official sources before you tap out`
4. `Save perks for later on your device`
5. `Get help, report outdated offers, and request new perks`

## Screenshot checklist

- use real seeded data
- avoid broken or placeholder states
- keep status bar clean
- use the same visual language across all screenshots
- prefer simple, high-contrast screens over crowded ones
