# Public URLs

Perk can publish its public legal and support pages through GitHub Pages.

## Expected URLs after deployment

If the repository is published through GitHub Pages using the current GitHub remote, the likely public URLs will be:

- Privacy Policy: `https://jk76761.github.io/PERK-/privacy`
- Support: `https://jk76761.github.io/PERK-/support`
- Help: `https://jk76761.github.io/PERK-/help`
- About: `https://jk76761.github.io/PERK-/about`

## Deployment path

1. Push the repository to `main`
2. Enable GitHub Pages in the repository settings
3. Let the `Deploy Web Pages` workflow publish the static build
4. Confirm the final live URLs before pasting them into App Store Connect

## Important note

If the repository name or Pages setup changes, the final URL may also change. Always verify the live page URL before submission.
