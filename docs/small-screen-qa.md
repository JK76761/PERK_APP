# Small-Screen QA Checklist

Last updated: 2026-05-18

Target first:

- iPhone SE size
- iPhone mini size
- any narrow-width iPhone in portrait

## Home

- [ ] hero text does not overlap the lime visual
- [ ] trust badge fits on one or two safe lines
- [ ] stats card labels do not overflow
- [ ] category labels stay inside their own area
- [ ] section headers do not collide with action pills

## Explore

- [ ] intro text wraps cleanly
- [ ] search bar remains aligned
- [ ] category chips stay scrollable and readable
- [ ] collection cards do not clip text
- [ ] perk cards remain readable with long titles

## Detail

- [ ] header actions stay aligned
- [ ] summary cards wrap safely
- [ ] region / eligibility metadata does not clip
- [ ] terms cards keep readable spacing
- [ ] similar perks do not overflow

## Saved

- [ ] empty state spacing looks intentional
- [ ] saved perk cards remain readable

## Profile / Preferences / Notifications

- [ ] settings rows do not push chevrons off-screen
- [ ] long descriptions wrap cleanly
- [ ] preference pills remain tappable and readable
- [ ] support rows remain aligned

## Functional checks

- [ ] share opens correctly
- [ ] email support links open correctly
- [ ] external provider links open correctly
- [ ] remote logos load when available
- [ ] initials fallback appears if a logo fails
